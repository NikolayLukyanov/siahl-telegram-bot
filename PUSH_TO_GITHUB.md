# Quick GitHub Setup

## Step 1: Create Repository
Go to: https://github.com/new

Fill in:
- Repository name: `siahl-telegram-bot`
- Description: `Telegram bot for SIAHL with AI features and analytics`
- Visibility: **Public**
- **DO NOT** check "Initialize this repository with a README"

Click "Create repository"

## Step 2: Push Code

Copy and paste these commands in your terminal:

```bash
cd "C:\Users\Lucas\Documents\GitHub\siahl-telegram-bot"
git remote add origin https://github.com/NikolayLukyanov/siahl-telegram-bot.git
git branch -M main
git push -u origin main
```

Done! Your code is now on GitHub.
